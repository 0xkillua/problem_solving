## Hints
single name A ,
 which A contains no more than 10 characters
 
## Solution python
```python
n1,n2 = sorted(map(int, input().split()))
print(n1+n2)
```
## Solution java

import java.util.Scanner;
 
public class Myproject {

       System.out.println(s + m);
        
    }
}
